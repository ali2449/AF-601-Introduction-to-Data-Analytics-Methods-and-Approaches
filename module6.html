np.array() - Create Array from Sequence

Basic Array Creation

The np.array() function is the most fundamental way to create NumPy arrays. It converts Python sequences (lists, tuples) into NumPy arrays.

numpy.array(object, dtype=None, *, copy=True, order='K', subok=False, ndmin=0)
Parameter Description Default
object Array-like object (list, tuple, etc.) Required
dtype Data type of elements None (inferred)
copy Whether to copy the input True
ndmin Minimum number of dimensions 0
Key Properties:
  • shape - Dimensions of the array
  • dtype - Data type of elements
  • ndim - Number of dimensions
  • size - Total number of elements

Creates an array of given shape and type, filled with ones. Useful for multiplicative operations and creating unit matrices.

np.empty() - Create Uninitialized Array

Fast Array Creation

Creates an array of given shape and type without initializing entries. This is the fastest way to create arrays when you plan to fill all values immediately.

numpy.empty(shape, dtype=float, order='C')
Important: The array contains garbage values from memory. Always fill the array completely before using it for computations.

Regular Sequence Arrays

Creates an array with evenly spaced values within a given interval. Similar to Python's range() but returns a NumPy array and works with floats.

numpy.arange([start,] stop[, step,], dtype=None)
Parameter Description Default
start Start of interval 0
stop End of interval (exclusive) Required

np.linspace() - Create Array with Specified Number of Points

Linearly Spaced Arrays

Creates an array with specified number of evenly spaced points between start and stop values. Unlike arange(), you specify the number of points rather than the step size.

numpy.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None, axis=0)

Real-World Application: In signal processing, linspace is used to create time arrays for sampling signals. In mathematical plotting, it creates smooth x-axis values for function visualization.

np.logspace() - Create Array with Logarithmically Spaced Points

Logarithmic Sequences

Creates an array with numbers spaced evenly on a log scale. Useful for data that spans multiple orders of magnitude.

numpy.logspace(start, stop, num=50, endpoint=True, base=10.0, dtype=None, axis=0)
Key Points:
  • start and stop are exponents of the base
  • Default base is 10, but can be changed
  • Useful for frequency ranges, probability distributions

np.diag() - Create Diagonal Array or Extract Diagonal

Diagonal Operations

Extract a diagonal or construct a diagonal array. Can work in two modes: extract diagonal from 2D array or create 2D array from 1D diagonal.

numpy.diag(v, k=0)
Dual Functionality:
  • If input is 1D: returns a 2D array with input on diagonal
  • If input is 2D: returns the diagonal as 1D array
  • k specifies which diagonal to use

np.reshape() - Change Array Shape

Reshaping Arrays

Gives a new shape to an array without changing its data. The total number of elements must remain the same.

numpy.reshape(a, newshape, order='C')
Key Points:
  • Returns a view when possible (same memory)
  • Use -1 to automatically calculate a dimension
  • Total elements must match original array

np.resize() - Change Array Size

Resizing Arrays

Returns a new array with the specified shape. If the new array is larger, the original array is repeated to fill the space.

numpy.resize(a, new_shape)
Important: Unlike reshape(), resize() can change the total number of elements and always returns a copy, not a view.

np.concatenate() - Join Arrays

Array Concatenation

Join a sequence of arrays along an existing axis. All arrays must have the same shape except in the dimension corresponding to the axis.

numpy.concatenate((a1, a2, ...), axis=0, out=None)

Real-World Application: In time series analysis, concatenate is used to combine data from different time periods. In image processing, it can stitch multiple images together.

Array Stacking

Join a sequence of arrays along a new axis. All input arrays must have the same shape.

numpy.stack(arrays, axis=0, out=None)
Key Difference from concatenate():
  • concatenate() - joins along existing axis
  • stack() - creates new axis and joins
  • All arrays must have identical shapes

Element-wise Operations

Operations on Array Elements

NumPy performs operations element by element, making it much faster than Python loops for numerical computations.

Key Advantages:
  • Vectorized operations (no loops needed)
  • Automatic broadcasting for compatible shapes
  • Significant performance improvements
  • Clean, readable code

Broadcasting

Operations Between Different Shapes

Broadcasting allows NumPy to perform operations on arrays of different shapes by automatically expanding the smaller array.

Broadcasting Rules:
  • Dimensions are compared from right to left
  • Dimensions must be equal or one of them must be 1
  • Missing dimensions are treated as 1
  • Arrays with compatible shapes can be broadcast together
