Section 1: Variables and Assignment

Understanding Variables

Variables are used to store data in Python. A variable is a named location in memory that holds a value, enhancing code reusability by allowing values to be referenced by name. Assignment is performed using the = operator.

Storing and Reusing Data

üìù Variable Naming Best Practices

  • Use descriptive names (monthly_sales vs ms)
  • Use lowercase with underscores (snake_case)
  • Start with a letter or underscore, not a number
  • Avoid Python keywords (print, if, for)
